Compass是一个基于SASS的类库，它帮助了我们预定义好了很多常用的Mixin和Function。

Compass的组成模块：

- CSS3：将CSS3的带有属性前缀的兼容代码组合成Mixin。
- Reset：用于清除浏览器自带样式，保证不同浏览器下显示的一致性。
- Utilities：对一些常用的CSS样式进行简化。
- Helpers：提供一些常用的函数。
- Layout：提供栅格系统和一些简单的布局样式。
